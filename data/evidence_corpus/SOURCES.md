# 증거 코퍼스 출처 안내 (Provenance)

`corpus.json` 의 스니펫은 **교육·데모 목적으로 손으로 작성한, 널리 알려진 공개
상식·과학적 합의의 요약문**입니다. 특정 기사·논문을 그대로 인용한 것이 아니며,
존재하지 않는 URL이나 구체 출처를 날조하지 않기 위해 `url` 은 모두 `null` 로,
`source` 는 정보의 **성격**(예: "과학 상식 정리", "보건·의학 일반 정보")으로
일반화해 표기했습니다.

## 신뢰도(credibility) 산정 방식
스니펫의 `source_type` 만 코퍼스에 저장하고, 실제 신뢰도 점수는 실행 시점에
`models.SOURCE_CREDIBILITY` 매핑으로 **결정론적으로** 부여됩니다.

| source_type | credibility |
|---|---|
| 정부·공공 | 0.90 |
| 학술 | 0.85 |
| 뉴스 | 0.70 |
| 기관 | 0.65 |
| 블로그·SNS | 0.40 |
| 출처불명 | 0.20 |

## 의도적으로 포함한 저신뢰 스니펫
`ev_vaccine_magnet_rumor` 등 일부 항목은 **출처불명(낮은 신뢰도)** 으로 두어,
판사 노드가 "출처 없는 주장을 감점"하고 "불충분(판단 불가)"을 정식 결론으로
사용하는 안전 설계가 동작하는지 검증할 수 있게 했습니다.
