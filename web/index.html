Rumor Verification Agent

검사·변호가 각자 전속 리서처로 증거를 수집해 3턴 토론을 벌이고, 판사가 자가 반박을 거쳐 LangGraph로 실시간 판정합니다.

● LIVE · 백엔드 연결 LangGraph 6노드 양측 전속 리서처 RAG 검사⇄변호 3턴 토론 자가 반박 루프

🧪 검증 입력

🖼 이미지 첨부 — 카톡 캡처·SNS 짤도 분석 입력창에 이미지를 드래그해도 됩니다.

🔒 키는 이 브라우저 세션에만 임시 보관되며, 서버에 저장·기록되지 않습니다(HTTPS 전송). 검증 비용은 입력하신 키로 청구됩니다.

📌 예시 입력 — 클릭하면 해당 문장으로 실시간 검증을 실행합니다.

※ 실제 LangGraph 에이전트가 동작하므로 한 번 검증에 수십 초가 걸릴 수 있습니다.
무료 호스팅 특성상 첫 접속은 서버 기동(콜드 스타트)으로 최대 1분쯤 더 걸릴 수 있습니다.
